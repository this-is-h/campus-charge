> ### 前言
>
> 你是否在下课后赶到充电桩却发现没有位置？想充电时却担心还有没有位置？现在，我们可以帮助你看到充电桩的实时状态！

这是一个服务于校园内电动车充电问题的项目，提供校园内所有充电桩的空闲情况及剩余时间，在有充电需求时提供参考。
